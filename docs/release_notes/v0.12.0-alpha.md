## New in this release

- **Clickhouse data connector**: Use Clickhouse as a data source.

- **Spark data connector**: Use Clickhouse as a data source.

- **Temporal refresh data period**: Constrain accelerated datasets to fetch data within a specified time window, ensuring you retain only the most recent time series data.

- **Spice Client SDK Support**: Use official Spice.ai SDK to connect to Spice runtime.
